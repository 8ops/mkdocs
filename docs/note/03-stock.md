# 股市秘法

## 一、网友经验1

1. 早盘急跌先别卖，封底加仓做T+0
2. 午后暴涨要减仓，午后暴跌次日进
3. 无冲高不卖，不跳水不买，横盘观望勿动
4. 十点涨停最硬气，安心持股别犹豫
5. 短线只做强势股，进退盯紧五日线
6. 红尘万丈，执手一心（勿贪多种方法）
7. 几度空仓几度红（行情差要空仓）
8. 低位单峰筹码线，主力来相见。



## 二、网友经验2

1. 大赚后要空仓，人的运气是有限
2. 连亏3天要果断卖出，肯定是买错了
3. 熊市买尾盘，熊市低点在尾盘
4. 牛市买早盘，牛市低点在早盘
5. 市场只有一个龙头，其他都是陪跑
6. 成交量最不骗人，天量意味着即将变盘
7. 连续下跌不抄底，因为没有底
8. 利好消息反着看，利空消息当利好
9. 三支标的就够了，太分散赚不到钱
10. 人这一生富一次就够了。
11. 趋势一旦形成，不会轻易改变。
12. 急涨慢跌是出货，急跌慢涨是吸筹
